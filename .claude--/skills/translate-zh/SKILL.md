# 中文翻译技能

## 术语表（必须遵守）
- `server` -> 服务器
- `client` -> 客户端
- `request` -> 请求
- `config` -> 配置
- `deprecated` -> 已弃用

## 规则
1. 只翻译注释和文档字符串，绝不修改代码逻辑。
2. 保留原有的缩进和换行。
3. 遇到专有名词（API名称、函数名）不翻译。
4. 中文和英文/数字之间加空格。
